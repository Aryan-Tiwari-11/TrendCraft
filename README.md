# TrendCraft - Full Stack eCommerce Website

## Overview
TrendCraft is a modern eCommerce web application built using the **MERN stack (MongoDB, Express, React, Node.js)**. This project allows users to explore products, filter and sort them, add items to the cart, and place orders using multiple payment methods including **Cash on Delivery**, **Stripe**, and **Razorpay**.  

The project also includes a comprehensive **Admin Dashboard** for managing products and orders.

---

## Features

### User Features
- Browse products by category and subcategory
- Filter and sort products by price, category, and type
- View product details and select product variants (like size or color)
- Add products to cart and update quantities
- Checkout using:
  - Cash on Delivery
  - Online Payment (Stripe and Razorpay)
- Order tracking and confirmation

### Admin Features
- Admin login and authentication
- Add new products with images, categories, and variants
- Update or delete existing products
- View all products and orders in a dashboard

---

## Technologies Used
- **Frontend:** React.js, Tailwind CSS, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Payment Gateways:** Stripe, Razorpay
- **Hosting/Deployment:** Vercel (Frontend), Node.js server can be hosted on any server
